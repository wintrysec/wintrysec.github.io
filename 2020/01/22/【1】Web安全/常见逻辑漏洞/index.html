гҖҗ1гҖ‘Webе®үе…Ё
2020-01-22

еёёи§ҒйҖ»иҫ‘жјҸжҙһ

  1. ж”Ҝд»ҳжјҸжҙһ
  2. и¶Ҡжқғи®ҝй—®
  3. дјҡиҜқеҠ«жҢҒ
    1. йҳІеҫЎж–№жі•
    2. SessionжңәеҲ¶
    3. CookieеұһжҖ§
  4. дјҡиҜқеӣәе®ҡ
    1. жјҸжҙһжЈҖжөӢ
    2. йҳІеҫЎж–№жі•

жҢ–жҺҳйҮҚзӮ№пјҡ

дёҡеҠЎжөҒзЁӢе’ҢHTTP/HTTPSиҜ·жұӮзҜЎж”№

ж”Ҝд»ҳжјҸжҙһе’Ңи¶ҠжқғжјҸжҙһжҳҜйҮ‘иһҚдёҡеҠЎдёӯеёёи§Ғзҡ„

ж”Ҝд»ҳжјҸжҙһ

(1) еҜҶз ҒйҮҚзҪ®

йӘҢиҜҒз ҒзӣҙжҺҘеңЁHTTPе“Қеә”дёӯиҝ”еӣһпјӣ

йӘҢиҜҒз ҒжңӘз»‘е®ҡз”ЁжҲ·пјҢжІЎе’ҢжүӢжңәеҸ·е’ҢйӮ®з®ұеҸ·еҒҡеҢ№й…ҚйӘҢиҜҒпјӣ

жңӘж ЎйӘҢз”ЁжҲ·еӯ—ж®өеҖјпјҢж”№иҮӘе·ұеҜҶз ҒпјҢжңҖеҗҺжҸҗдәӨе…¶е®ғUIDпјӣ

йӘҢиҜҒз ҒдёҚеӨұж•ҲпјҢеҸҜжһҡдёҫпјӣ

(2) ж”Ҝд»ҳи®ўеҚ•

зҜЎж”№ж”Ҝд»ҳйҮ‘йўқпјҢиҝҗиҙ№дҝ®ж”№дёәиҙҹж•°пјҢдҪҝжҖ»йҮ‘йўқйҷҚдҪҺгҖӮ

(3) з«һдәүжқЎд»¶

еңЁж–Үд»¶дёҠдј дёӯе’Ңиҙӯзү©ж—¶пјӣ

Aз”ЁжҲ·дҪҷйўқ10е…ғпјҢBе•Ҷе“Ғ5е…ғпјҢCе•Ҷе“Ғ6е…ғ;

AеҲ©з”Ёз«һдәүжқЎд»¶еӨҡзәҝзЁӢеҗҢж—¶еҸ‘иө·иҙӯд№°Bе’ҢCзҡ„иҜ·жұӮпјӣ

еҸҜиғҪзҡ„з»“жһңжңүпјҡ

жңүдёҖд»¶е•Ҷе“Ғиҙӯд№°еӨұиҙҘпјӣ

е•Ҷе“ҒйғҪиҙӯд№°жҲҗеҠҹпјҢдҪҶеҸӘжүЈдәҶ6е…ғпјӣ

е•Ҷе“ҒйғҪиҙӯд№°жҲҗеҠҹпјҢдҪҶдҪҷйўқдёә -1е…ғпјӣ

и¶Ҡжқғи®ҝй—®

и¶Ҡжқғи®ҝй—®д»–дәәдҝЎжҒҜжҲ–ж“Қзәөд»–дәәиҙҰеҸ·

ж°ҙе№іи¶Ҡжқғпјҡ

еҗҢзә§еҲ«(жқғйҷҗ)з”ЁжҲ·д№Ӣй—ҙпјҢи¶Ҡжқғи®ҝй—®йқһжі•ж“Қзәөе…¶е®ғиҙҰжҲ·пјӣ(иҝҷдјҡеҜјиҮҙеӨ§жү№йҮҸж•°жҚ®жі„йңІпјҢжҒ¶ж„ҸзҜЎж”№)

еһӮзӣҙи¶Ҡжқғпјҡ

дёҚеҗҢзә§еҲ«д№Ӣй—ҙзҡ„з”ЁжҲ·и¶ҠжқғпјҢжҷ®йҖҡз”ЁжҲ·жү§иЎҢз®ЎзҗҶе‘ҳзҡ„еҠҹиғҪпјӣ

и¶Ҡжқғи®ҝй—®ж”»еҮ»жөӢиҜ•пјҡ

и¶Ҡжқғи®ҝй—®дҝ®еӨҚе»әи®®пјҡ

и¶Ҡжқғи®ҝй—®жјҸжҙһзҡ„дё»иҰҒеҺҹеӣ жҳҜжІЎжңүеҜ№з”ЁжҲ·зҡ„иә«д»ҪеҒҡеҲӨж–ӯе’ҢжҺ§еҲ¶пјҢйҳІжҠӨиҝҷз§ҚжјҸжҙһеҸҜд»ҘйҖҡиҝҮsessionжқҘжҺ§еҲ¶гҖӮ

з”ЁжҲ·зҷ»еҪ•жҲҗеҠҹеҗҺпјҢжҠҠusernameе’ҢUIDзӯүдҝЎжҒҜеҶҷе…ҘеҲ°sessionдёӯпјҢ

еҪ“жҹҘзңӢдёӘдәәдҝЎжҒҜж—¶пјҢд»ҺsessionдёӯеҸ–еҮәusernameпјҢиҖҢдёҚжҳҜд»ҺGETе’ҢPOSTеҸ–пјҢжӯӨж—¶usernameе°ұжҳҜжІЎиў«зҜЎж”№зҡ„гҖӮ

дјҡиҜқеҠ«жҢҒ

дјҡиҜқеҠ«жҢҒпјҲSession hijackingпјүпјҢиҝҷжҳҜдёҖз§ҚйҖҡиҝҮиҺ·еҸ–з”ЁжҲ· Session ID еҗҺпјҢдҪҝз”ЁиҜҘ Session ID зҷ»еҪ•зӣ®ж ҮиҙҰеҸ·зҡ„ж”»еҮ»ж–№жі•пјҢжӯӨж—¶ж”»еҮ»иҖ…е®һйҷ…дёҠжҳҜдҪҝз”ЁдәҶзӣ®ж ҮдјҡиҜқеӣәе®ҡжјҸжҙһеҹәжң¬йҳІеҫЎж–№жі• иҙҰжҲ·зҡ„жңүж•Ҳ SessionгҖӮ

дјҡиҜқеҠ«жҢҒзҡ„з¬¬дёҖжӯҘжҳҜеҸ–еҫ—дёҖдёӘеҗҲжі•зҡ„дјҡиҜқж ҮиҜҶжқҘдјӘиЈ…жҲҗеҗҲжі•з”ЁжҲ·пјҢеӣ жӯӨйңҖиҰҒдҝқиҜҒдјҡиҜқж ҮиҜҶдёҚиў«жі„жјҸгҖӮ

еҸ—е®іиҖ…зҷ»еҪ•з«ҷзӮ№пјҢжңҚеҠЎеҷЁиҝ”еӣһдёҖдёӘдјҡиҜқж ҮиҜҶ(Session ID)

й»‘е®ўжҚ•иҺ·иҝҷдёӘSession IDпјҲ зҪ‘з»ңе—…жҺўпјҢXSS пјүпјҢдҪҝз”ЁиҝҷдёӘSession IDи®ҝй—®з«ҷзӮ№иҺ·еҫ—еҸ—е®іиҖ…еҗҲжі•дјҡиҜқ

йҳІеҫЎж–№жі•

XSSжјҸжҙһеј•иө·зҡ„дјҡиҜқеҠ«жҢҒпјҡдҪҝз”Ёhttp-onlyжқҘйҳІжӯўJSиҺ·еҸ–cookieдёӯзҡ„Session IDдҝЎжҒҜ

зҪ‘з»ңе—…жҺўеј•иө·зҡ„дјҡиҜқеҠ«жҢҒпјҡдҪҝз”ЁHTTPS+secureжқҘдҝқиҜҒSession IDдёҚиў«е—…жҺўиҺ·еҸ–еҲ°

SessionжңәеҲ¶

SessionжңәеҲ¶жҳҜдёҖз§ҚжңҚеҠЎеҷЁз«Ҝзҡ„жңәеҲ¶пјҢжңҚеҠЎеҷЁдҪҝз”ЁдёҖз§Қзұ»дјјдәҺж•ЈеҲ—иЎЁзҡ„з»“жһ„жқҘдҝқеӯҳдҝЎжҒҜз”ЁдәҺдҝқжҢҒзҠ¶жҖҒгҖӮ

дҝқеӯҳиҝҷдёӘSession IDжңҖдёәж–№дҫҝзҡ„ж–№ејҸжҳҜйҮҮз”ЁCookieгҖӮ

Cookieзҡ„еҗҚеӯ—йғҪжҳҜзұ»дјјдәҺSESSIONIDпјӣ

weblogicеҜ№дәҺwebеә”з”ЁзЁӢеәҸз”ҹжҲҗзҡ„cookieпјҢJSESSIONIDпјӣ

PHPдёӯSessionзҡ„й»ҳи®ӨеҗҚз§°жҳҜPHPSESSIDгҖӮ

CookieеұһжҖ§

HttpOnly и®ҫзҪ®ж–№жі•

жңҚеҠЎз«ҜеҸ‘йҖҒcookieзҡ„ж—¶еҖҷпјҢеҸҜд»Ҙи®ҫзҪ®HTTP-Only пјҢзҰҒжӯў JS иҺ·еҸ–CookieеҶ…е®№

Secure

и®ҫзҪ®cookieзҡ„жҹҗдёӘеҖјsecureдёәTrueж—¶пјҢ жӯӨcookieеҸӘжңүеңЁHTTPSеҚҸи®®дёӯжүҚдјҡиҝӣиЎҢдј иҫ“

HTTPеҚҸи®®дј иҫ“ж—¶пјҢжҳҜдёҚдј иҫ“жӯӨеҚҸи®®зҡ„гҖӮ

дјҡиҜқеӣәе®ҡ

дјҡиҜқеӣәе®ҡпјҲSession fixationпјүжҳҜдёҖз§ҚиҜұйӘ—еҸ—е®іиҖ…дҪҝз”Ёж”»еҮ»иҖ…жҢҮе®ҡзҡ„дјҡиҜқж ҮиҜҶпјҲSession IDпјүзҡ„ж”»еҮ»жүӢж®өгҖӮиҝҷжҳҜж”»еҮ»иҖ…иҺ·еҸ–еҗҲжі•дјҡиҜқж ҮиҜҶзҡ„жңҖз®ҖеҚ•зҡ„ж–№жі•гҖӮдјҡиҜқеӣәе®ҡд№ҹеҸҜд»ҘзңӢжҲҗжҳҜдјҡиҜқеҠ«жҢҒзҡ„дёҖз§Қзұ»еһӢпјҢеҺҹеӣ жҳҜдјҡиҜқеӣәе®ҡзҡ„ж”»еҮ»зҡ„дё»иҰҒзӣ®зҡ„еҗҢж ·жҳҜиҺ·еҫ—зӣ®ж Үз”ЁжҲ·зҡ„еҗҲжі•дјҡиҜқгҖӮдёҚиҝҮдјҡиҜқеӣәе®ҡиҝҳеҸҜд»ҘжҳҜејәиҝ«еҸ—е®іиҖ…дҪҝз”Ёж”»еҮ»иҖ…и®ҫе®ҡзҡ„дёҖдёӘжңүж•ҲдјҡиҜқпјҢд»ҘжӯӨжқҘиҺ·еҫ—з”ЁжҲ·зҡ„ж•Ҹж„ҹдҝЎжҒҜгҖӮ

  • и®ҝй—®зҪ‘з«ҷж—¶пјҢзҪ‘з«ҷдјҡи®ҫзҪ®cookieдёӯзҡ„Session ID
  • еҪ“з”ЁжҲ·зҷ»еҪ•еҗҺпјҢcookieдёӯзҡ„SessionIDдҝқжҢҒдёҚеҸҳпјҲеҪўжҲҗеҺҹеӣ пјү
  • еҸӘиҰҒиҺ·еҸ–зҷ»йҷҶеүҚзҡ„Session IDеҶ…е®№пјҢе°ұеҸҜд»ҘзҹҘйҒ“зҷ»йҷҶеҗҺзҡ„Session ID
  • й»‘е®ўз”ЁиҜҘSession IDжһ„йҖ й“ҫжҺҘпјҢеҸ‘йҖҒз»ҷеҸ—е®іиҖ…зӮ№еҮ»еҗҺпјҢй»‘е®ўжҲҗеҠҹеҠ«жҢҒеҸ—е®іиҖ…зҡ„дјҡиҜқ

жјҸжҙһжЈҖжөӢ

и®ҝй—®зҪ‘з«ҷпјҲжңӘзҷ»еҪ•пјүпјҡиҺ·еҸ–cookieдҝЎжҒҜпјҢиҺ·еҸ–Session ID

зҷ»еҪ•зҪ‘з«ҷпјҡжҹҘзңӢCookieдҝЎжҒҜпјҢиҺ·еҸ–Session ID

жҹҘзңӢзҷ»еҪ•еүҚпјҢзҷ»еҪ•еҗҺSessionIDжҳҜеҗҰзӣёеҗҢ

йҳІеҫЎж–№жі•

1гҖҒеңЁз”ЁжҲ·зҷ»еҪ•жҲҗеҠҹеҗҺйҮҚж–°еҲӣе»әдёҖдёӘSession IDпјҢдҪҝзҷ»еҪ•еүҚзҡ„еҢҝеҗҚдјҡиҜқејәеҲ¶еӨұж•Ҳ

2гҖҒSessionIDдёҺжөҸи§ҲеҷЁз»‘е®ҡпјҡSessionIDдёҺжүҖи®ҝй—®жөҸи§ҲеҷЁжңүеҸҳеҢ–пјҢз«ӢеҚійҮҚзҪ®

3гҖҒSessionIDдёҺжүҖи®ҝй—®зҡ„IPз»‘е®ҡпјҡSessionIDдёҺжүҖи®ҝй—®IPжңүеҸҳеҢ–пјҢз«ӢеҚійҮҚзҪ®
