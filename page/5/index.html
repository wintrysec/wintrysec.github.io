会话管理漏洞

会话劫持

原理

会话劫持（Session hijacking），这是一种通过获取用户 Session ID 后，使用该 Session ID 登录目标账号的攻击方法，

此时攻击者实际上是使用了目标会话固定漏洞基本防御方法 账户的有效 Session。

会话劫持的第一步是取得一个合法的会话标识来伪装成合法用户，因此需要保证会话标识不被泄漏。
